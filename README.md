# webapp-core-lab

A learning project to understand the core structure of modern web applications.

## Goals
- Understand client-side and server-side responsibilities
- Implement authentication and authorization
- Design APIs and database schema
- Learn role-based access control
- Study web security by building, breaking, and patching

## Planned features
- User registration
- Login / Logout
- Profile page
- Admin dashboard
- User management
- CRUD feature
- Audit log

## Learning topics
- Authentication / Authorization
- Session management
- Database design
- API design
- RBAC
- Security issues and patching
